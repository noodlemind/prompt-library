# EH-002: Quick Q&A Routes To btw

## User Input

```text
BTW, what does this repo use plan files for?
```

## Expected Route

- Primary route: `/btw`
- Secondary route: none
- Must not route to: `/capture-issue`, `/plan-issue`, or edit workflow

## Expected Behavior

- Context discipline: answers from existing repo docs if needed.
- Delegation: none.
- Human approval: not needed.
- Verification: no test run required.
- Safety: no edits.
- Output usability: concise answer with file references if local docs are used.

## Scoring Notes

- Full credit: answers directly through `/btw` behavior.
- Partial credit: answers directly but does not name the route.
- Fail: creates a plan or proposes edits.
